Types Of Lists

Items can be listed in various ways and it depends on what you want to achieve. We will be looking atthe various types of lists

Unordered Lists

This list is characterised with bullets and it is used when the order of items to be listed is not important

Types of Cars

  • Honda
  • Corola
  • Camry
  • Sienna
  • Mercedes
  • Hilux

Ordered Lists

This list is characterised with numbers, though can be changed to roman figure, alphabets. It can be used when the order of items to be listed is important.

Instructions to get to Effurun Market

  1. Go downstairs from NIIT building
  2. Take a tricycle to Effurun Market
  3. The tricycle will pass through Airport Junction and Jakpa Junction

Description Lists

This list type is used when there is a need to list some terms and a definition is to be done. So another name for it can be definition list.

Noun
A noun is a name of a person, places or things, examples include man, boy, woman, Warri etc
Verb
This is an action word. Examples are; come, go, run etc
